To use the Trading Symbols skill, try saying...

* *Alexa, start trading symbols*

* *i don't know this one*

* *Alexa play trading symbols*

Alexa gives user a ticker symbol for a public company that trades on US exchanges and asks for the name of the company represented by the ticker symbol.
