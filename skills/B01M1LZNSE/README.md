To use the Earth Object Watcher skill, try saying...

* *Alexa, ask Earth Object Watcher for the number of near earth objects.*

* *How many near earth objects are there*

* *How many close approach objects are there*

This skill tells the number of near earth objects or close approach objects. The data is retrieved from NASA JPL NEO's open API.
