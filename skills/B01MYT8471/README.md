To use the Christmas Countdown skill, try saying...

* *Alexa, ask Santa Countdown how long until Christmas*

* *Alexa, ask Santa Countdown how many days until Christmas*

This is just a little skill that tells you how many days until the next Christmas. It can be prompted using the keyword 'Santa Countdown', which makes it fun for all. Be sure to check for the special message on Christmas day!
