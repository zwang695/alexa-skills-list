To use the Encouraging Quote skill, try saying...

* *Alexa Ask Encouraging Quote*

* *Alexa Ask Encouraging Quote for one more*

* *Alexa Open Encouraging Quote*

If you open the application, it will provide an Inspirational Quote. If you need one more quote you just say "one more" or "quote"
