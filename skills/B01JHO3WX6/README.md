To use the USPopulation skill, try saying...

* *Alexa, ask U.S. Population today*

* *Alexa, ask U.S. Population end of year*

* *Alexa, ask U.S. Population today's population*

This skill will provide the us population for today or end of last year.  The data source is provided by api.population.io.
